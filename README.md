# Reimplementation of "Hypocoercivity properties of adaptive Langevin dynamics" by Leimkuhler, Sachs and Stoltz

[Hypocoercivity properties of adaptive Langevin dynamics](https://arxiv.org/abs/1908.09363)

The goal of this project of to reproduce the numerical experiments of the article

Produced as a homework for the class "Introduction to Computational Statistical Physics", taught by Gabriel Stoltz in the Mathematical Modelisation Master at Sorbonne University
